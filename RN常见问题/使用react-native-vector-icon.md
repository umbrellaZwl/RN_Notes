## 在React Native中使用[react-native-vector-icon](https://github.com/oblador/react-native-vector-icons)

直接放两个链接，讲得很全也很仔细

* [react-native-vector-icons的简单使用,图片,按钮,标签视图,导航条](http://www.cnblogs.com/moxiaoyan33/p/5482024.html)
* [React Native react-native-vector-icons 使用小记](http://blog.csdn.net/uuq/article/details/53150841)

自己简单总结一下：
```bash
+ npm install react-native-vector-icons --save #(回车)
+ npm install rnpm -g
+ rnpm link #(回车) 
```

如果可能，至此里面的icon已经能用了
但是大多数需要用xcode打开工程进行一下操作

+ (1).右键工程文件Add Files to "(你工程名)"
+ (2).选择node_modules/react-native-vector-icons/Fonts文件
+ (3).点击"完成".

至此icon如果还不能使用，那么你人品有问题!
